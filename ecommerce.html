Search Engine Optimisation

Search Engine Optimisation (SEO) refers to the practice of driving users to a website from organic search results. For example, just because you have an ecommerce store it does not mean your products are found in Google when a user searches for them. Instead, a user would first have to navigate to your store to find them.

SEO is almost considered a dark art, as nobody knows exactly what algorthims run the Google search engine. This has lead to a wide variety of SEO tactics and strategies, from content generation to link building. I deal with technical SEO, working to fix the basics first and build from the ground up. From my experience, this is the best approach for an ecommerce store, working to increase the number of indexed products and as such sales.

Site Audits

Using a paid software, I conduct website audits by crawling ecommerce sites to assess overall website health. After a site audit is complete I can outline where customers can improve basic elements to increase their chances of being indexed on Google. Furthermore, I am happy to help implement changes where possible.

Google Tools

There are a number of SEO tools all website owners should have. Two of the main being Google Analytics and Google Search Console. I can help install these tools on clients websites.
